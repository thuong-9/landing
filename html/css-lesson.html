Giới thiệu CSS

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để tạo phong cách và định dạng cho các phần tử được xây dựng bằng HTML. Nếu HTML đóng vai trò xây dựng cấu trúc trang web (tạo đoạn văn, tiêu đề, bảng,…), thì CSS giúp tô điểm giao diện như thay đổi màu sắc, bố cục, font chữ, kích thước và hình dạng của các phần tử đó.


CSS được phát triển bởi W3C (World Wide Web Consortium) vào năm 1996, nhằm bổ sung khả năng định dạng cho HTML - vốn không được thiết kế để trình bày giao diện phức tạp.


Cách hoạt động của CSS là dựa vào vùng chọn (selector) - có thể là tên thẻ HTML, ID, class,… rồi áp dụng các thuộc tính style tương ứng cho vùng đó.


Giữa HTML và CSS tồn tại mối quan hệ chặt chẽ: HTML xây dựng khung nội dung, còn CSS tạo nên diện mạo - hai ngôn ngữ này không thể tách rời trong thiết kế web.


Vai trò của CSS

Nếu HTML cung cấp cấu trúc và nội dung cho trang web (bộ xương), thì CSS chịu trách nhiệm cho kiểu dáng, màu sắc, bố cục và hiệu ứng (trang phục và vẻ ngoài).

  • Quyết định màu chữ, màu nền, font chữ.
  • Kiểm soát vị trí của các phần tử (layout).
  • Tạo hiệu ứng động và giao diện người dùng tương tác.
  • Giúp trang web hiển thị đẹp và tối ưu trên nhiều thiết bị (Responsive Design).

Tại sao CSS lại quan trọng?

  • Tiết kiệm thời gian: Có thể định dạng hàng trăm trang HTML chỉ bằng cách thay đổi một tệp CSS duy nhất.
  • Phân tách nhiệm vụ: Tách biệt cấu trúc (HTML) và trình bày (CSS), giúp code dễ đọc, dễ bảo trì và cập nhật.
  • Trải nghiệm người dùng tốt hơn: Tạo ra các trang web có giao diện thu hút và chuyên nghiệp.

Cú pháp CSS


CSS được tạo thành từ các Style Rule. Một rule gồm 3 phần:

selector { property: value }

Giải thích các thẻ:

  • Selector: Bộ chọn phần tử HTML (ví dụ: div, p, span,...)
  • Property: Thuộc tính CSS như color, border,...
  • Value: Giá trị của thuộc tính (ví dụ: red, #fff,...)

Giải thích các thẻ:

  • Trong ví dụ trên, selector trỏ tới phần tử HTML là h1. Đây là phần tử mà mình muốn tạo style cho nó.
  • Khối khai báo Declaration có thể bao gồm một hoặc nhiều khối và được đặt phân biệt nhau bởi dấu ";".
  • Mỗi khối khai báo này bao gồm một tên của CSS property (trong ví dụ là color và font-size) và một giá trị (trong ví dụ là blue và 12px) được gán cho property đó.
  • Bạn cần chú ý một điều rằng, một khai báo CSS luôn luôn kết thúc với một dấu chấm phẩy và phần khai báo sẽ được bao quanh bởi cặp dấu {}.
Ví dụ 2:
p { color: red; text-align: center; }

Giải thích các thẻ:

  • p là một bộ chọn trong CSS (nó trỏ đến phần tử HTML mà bạn muốn định dạng: <p>).
  • color là một thuộc tính và redlà giá trị thuộc tính.
  • text-align là một thuộc tính và centerlà giá trị thuộc tính

Bộ chọn CSS


Bộ chọn CSS được sử dụng để "tìm" (hoặc chọn) các phần tử HTML mà bạn muốn định kiểu.

Chúng ta có thể chia bộ chọn CSS thành năm loại:

  • Bộ chọn đơn giản.
  • Bộ chọn kết hợp.
  • Bộ chọn lớp giả.
  • Bộ chọn phần tử giả.
  • Bộ chọn thuộc tính.

Bộ chọn đơn giản

Bộ chọn đơn giản chọn các phần tử dựa trên tên phần tử, ID và lớp.

Selector Example Example description
element p Selects all <p> elements
#id #firstname Selects the element with id="firstname"
* * Selects all elements
.class .intro
p.intro		 Selects all elements with class="intro"
Selects all <p> elements with class="intro"

Bộ chọn kết hợp

Bộ kết hợp là dấu hiệu giải thích mối quan hệ giữa các bộ chọn.

Name Sign Example Example Result
Child combinator > div > p Selects every <p> element that are direct children of a <div> element
Descendant combinator (single space) div p Selects all <p> elements inside <div> elements
Namespace separator | ns | h2 Selects all <h2> elements in namespace ns
Next-sibling combinator + div + p Selects the first <p> element that is placed immediately after <div> elements
Selector list , div, p Selects all <div> elements and all <p> elements
Subsequent-sibling combinator ~ p ~ ul Selects all <ul> elements that are preceded by a <p> element

Bộ chọn lớp giả

Lớp giả CSS là một từ khóa có thể được thêm vào bộ chọn để xác định kiểu cho trạng thái đặc biệt của một phần tử.

Một số cách sử dụng phổ biến cho lớp giả:

  • Tạo kiểu cho một phần tử khi người dùng di chuyển chuột qua nó.
  • Kiểu liên kết đã truy cập và chưa truy cập khác nhau.
  • Tạo kiểu cho một phần tử khi nó được chú ý.
  • Kiểu các thành phần biểu mẫu hợp lệ/không hợp lệ/bắt buộc/tùy chọn.
  • Định dạng một phần tử là phần tử con đầu tiên của phần tử cha của nó.

Cú pháp: Các lớp giả luôn được biểu thị bằng dấu hai chấm (:) theo sau là tên lớp giả

selector:pseudo-class-name { CSS properties }

Bộ chọn thuộc tính

Bộ chọn thuộc tính sẽ chọn các phần tử HTML có tập hợp thuộc tính nhất định.

Selector Example Example description
[attribute] [lang] Selects all elements with a lang attribute
[attribute=value] [lang="it"] Selects all elements with lang="it"
[attribute~=value] [title~="flower"] Selects all elements with a title attribute containing the word "flower"
[attribute|=value] [lang|="en"] Selects all elements with a lang attribute value equal to "en" or starting with "en-"
[attribute^=value] [href^="https"] Selects all elements with a href attribute value that begins with "https"
[attribute$=value] [href$=".pdf"] Selects all elements with a href attribute value ends with ".pdf"
[attribute*=value] [href*="w3schools"] Selects all elements with a href attribute value containing the substring "w3schools"

Cách thực hiện CSS


Khi trình duyệt đọc một bảng định kiểu, nó sẽ định dạng tài liệu HTML theo thông tin trong bảng định kiểu đó.

Chúng ta sẽ có ba cách sử dụng CSS trong HTML là:

  • Inline: mã CSS viết tại thuộc tính Style của phần tử HTML
  • Internal: mã CSS trong chính văn bản HTML, nằm trong khối thẻ <style>
  • External: mã CSS ở một file riêng biệt sau đó nạp vào trong HTML bằng phần tử <link>

Inline CSS

Đặt thuộc tính style vào thẻ mở của phần tử HTML, giá trị của thuộc tính style là các cặp thuộc tính định dạng CSS.

Ví dụ

<!DOCTYPE html>

<html>
<body>
    <img src="http://webcoban.vn/image/h2.jpg" style="width:200px;height:100px">
</body>
</html>

Internal CSS

  • Ta đặt các cặp thuộc tính định dạng CSS vào bên trong cặp thẻ <style><type=”text/css”></style>
  • Cặp thẻ <style type=”text/css”></style> thì được đặt bên trong cặp thẻ <head></head>

Ví dụ

<!DOCTYPE html>
<html>
<head>    
    <style type=”text/css”>        
        h1 { font-size:30px; color:white; background-color:red;}        
        p { background-color:gray; }    
    </style>
</head>
<body>    
    <h1>webcoban.vn (WEB CƠ BẢN .VN)</h1>    
    <p>Hướng dẫn học lập trình web từ cơ bản đến nâng cao</p>
</body>
</html>

External CSS

  • Với External CSS, ta đặt các thuộc tính định dạng vào bên trong tập tin CSS.
  • Trong cặp thẻ <head></head> của tập tin HTML, ta dùng thẻ sau để nhúng tin CSS vào trang web.
    <link rel=”stylesheet’type=”text/css”href=”đường dẫn đến tập tin CSS”>

Ví dụ

Có một tập tin dinhdang.css (cùng cấp với tập tin HTML) có nội dung là:
    h1 { color:red; font-size:50px; font-family:Courier;}
                            
Khi thực thi tập tin HTML có nội dung:
<!DOCTYPE html>
    <html>    
        <head>        
            <link rel=”stylesheet” type=”text/css” href=”dinhdang.css”>    
        </head> 
        <body>>       
            <h1>Hello HTML</h1>    
        </body>
    </html>

Thứ tự ưu tiên các cách sử dụng CSS

Khi một thuộc tính nào đó của phần tử được thiết lập bởi nhiều cách, thì thuộc tính sẽ nhận giá trị của các cách sử dụng CSS theo độ ưu tiên như sau:

  1. Inline CSS (ưu tiên cao nhất)
  2. Internal CSS
  3. External CSS (ưu tiên thấp nhất)

Comment trong CSS


  • Bình luận được sử dụng để giải thích mã CSS và có thể hữu ích khi bạn chỉnh sửa mã nguồn sau này.
  • Bình luận cũng được sử dụng để tạm thời vô hiệu hóa các phần mã CSS trong bảng định kiểu.
  • Trình duyệt sẽ bỏ qua các bình luận!
  • Bình luận CSS được đặt bên trong phần tử <style> của HTML và bắt đầu bằng /* và kết thúc bằng */

Ví dụ 1:

/* This is a single-line comment */
    p {
    color: red;
}

Ví dụ 2: Bạn có thể thêm bình luận về bất cứ đâu bạn muốn trong mã.

p {
color: red;  /* Set text color to red */
}

Ví dụ 3: Ngay giữa một dòng mã.

p {
  color: /*red*/blue; 
}

Ví dụ 4: Bình luận cũng có thể kéo dài nhiều dòng

/* This is
a multi-line
comment */

p {
  color: red;
}

Màu sắc



Trong CSS, màu sắc được chỉ định bằng cách sử dụng tên màu được xác định trước hoặc bằng giá trị RGB, HEX, HSL, RGBA, HSLA.

Mã màu RGB

Màu sắc trong CSS có thể được định nghĩa bằng mã màu RGB (R = Red, G = Green và B= Blue). Nói cách khác chúng ta có thể tạo ra tất cả các màu sắc khác từ việc pha trộn 3 màu đỏ (RED), xanh lá cây (Green) và xanh da trời (Blue). Cú pháp giá trị màu RGB trong CSS:

rgb(red, green, blue)

Trong đó giá trị của các màu red, green và blue dao động từ 0 tới 255, thể hiện độ độ đậm của chúng để hình thành nên màu sắc cuối cùng (cao nhất là 255 và thấp nhất là 0).

Ví dụ:

Giá trị rgb(255,0,0) là màu đỏ vì màu đỏ có giá trị cao nhất là 255 còn xanh lá cây và xanh da trời là 0.


Mã màu HEX

Trong CSS cũng có một cách khác để chỉ định màu sắc đó chính là giá trị HEX. Hex (Hexadecimal) là một trong những phương pháp định nghĩa màu sắc được sử dụng phổ biến nhất trên web. Phương pháp này sử dụng các số trong hệ cơ số 16 để thể hiện giá trị của màu sắc.

Cú pháp giá trị màu HEX trong CSS:

#rrggbb

Trong đó:

- rr (red), gg (green) và bb (blue) tương ứng là thành phần màu đỏ, xanh lá cây và xanh da trời. Nó được biểu diễn bởi các giá trị trong hệ thập lục phân (tương đương 0 - 255 trong hệ thập phân).

- Ở cơ số 16, giá trị mỗi màu trong ba màu trên dao động từ 00 (không có màu) tới FF (full color).

Ví dụ:

#ffffff đại diện cho màu trắng.

#000000 đại diện cho màu đen.

#ff0000 đại diện cho màu đỏ.


Mã màu HSL

HSL là chữ cái đầu viết tắt của Hue, Saturation và Lightness, ký hiệu của nó là hsl(hue, saturation, lightness).

Trong đó:

  • Hue là độ của màu trong vòng tròn màu có giá trị từ 0 đến 360 (0 là màu đỏ, 120 là màu xanh lá, 240 là màu xanh da trời)
  • Saturation có giá trị phần trăm (%) cường độ màu, cao nhất là 100% (full)
  • Lightness cũng có giá trị phần trăm, là độ sáng của màu theo phần trăm, 0% là tối và 100% là sáng trắng.

hsl(300, 100%, 10%)
hsl(300, 100%, 20%)
hsl(300, 100%, 30%)
hsl(300, 100%, 40%)
hsl(300, 100%, 50%)
hsl(300, 100%, 60%)
hsl(300, 100%, 70%)
hsl(300, 100%, 80%)
hsl(300, 100%, 90%)
hsl(300, 100%, 100%)

Mã màu RGBA

RGBA color là phần mở rộng của giá trị màu RGB với thành phần bổ sung là chỉ số opacity quy định độ mờ/độ trong suốt của màu sắc.

rgba(red, green, blue, alpha)

Trong đó:

alpha là chỉ số opacity có giá trị từ 0.0 - 1.0, giá trị càng nhỏ thì độ trong suốt càng nhiều.


rgba(88,37,123, 0.1)
rgba(88,37,123, 0.2)
rgba(88,37,123, 0.3)
rgba(88,37,123, 0.4)
rgba(88,37,123, 0.5)
rgba(88,37,123, 0.6)
rgba(88,37,123, 0.7)
rgba(88,37,123, 0.8)
rgba(88,37,123, 0.9)
rgba(88,37,123, 1.0)

Mã màu HSLA

HSLA Color là phần mở rộng của giá trị màu HSL với thành phần bổ sung là chỉ số opacity quy định độ mờ/độ trong suốt của màu sắc.

hsla(hue, saturation, lightness, alpha)

Trong đó:

alpha là chỉ số opacity có giá trị từ 0.0 - 1.0, giá trị càng nhỏ thì độ trong suốt càng nhiều.

Nền


Đặc tính nền trong CSS được dùng để định nghĩa hiệu ứng nền cho các phần tử. Các đặc tính của nền trong CSS gồm:

  • Thuộc tính background-color: Sử dụng để thiết lập màu nền của một phần tử.
  • Thuộc tính background-image: Sử dụng để thiết lập hình nền cho một phần tử.
  • Thuộc tính background-repeat: Sử dụng để điều khiển sự lặp đi lặp lại của một hình ảnh nền theo chiều dọc hoặc chiều ngang.
  • Thuộc tính background-position: Sử dụng để điều khiển vị trí của một hình ảnh nền.
  • Thuộc tính background-attachment: Sử dụng để xác định xem có hay không một hình nền là cố định hoặc có thể scroll với phần còn lại của trang.

Màu nền

Đặc tính backgroud-color xác định màu nền cho phần tử.

body { background-color: lightblue;}

Màu sắc trong CSS thường được xác định bằng

  • Tên màu, như “red”...
  • Giá trị màu HEX, như #ff0000…
  • Giá trị RGB, như rgb(255,0,0)...

Ví dụ: các phần tử <h1>, <p> và <div> có màu nền khác nhau.

h1 { background-color: green;}
div { background-color: lightblue;}
p { background-color: yellow;}

Ảnh nền

Đặc tính background-image lấy hình ảnh làm nền cho phần tử. Mặc định ảnh sẽ được sao chép lặp lại để bao phủ toàn bộ phần tử.

body { background-image: url("paper.gif");}

Ảnh nền lặp đi lặp lại theo chiều dọc và ngang

Mặc định đặc tính background-image sẽ lặp đi lặp lại theo cả chiều ngang và chiều dọc để bao phủ toàn bộ phần tử. Một số ảnh chỉ nên lặp theo 1 chiều, nếu không trông sẽ rất xấu. Khi đó hãy dùng thêm đặc tính background-repeat: repeat-x; để lặp theo chiều ngang và background-repeat: repeat-y; để lặp theo chiều dọc.

body { background-image: url("gradient_bg.png"); background-repeat: repeat-x;}

Ảnh nền, đặt vị trí và không lặp lại

Để lấy một ảnh làm nền và không muốn lặp lại thì dùng đặc tính background-repeat.

body { background-image: url("img_trejpg"); background-repeat: no-repeat;}

Ảnh nền, cố định vị trí

Để chỉ định ảnh nền cố định (không bị cuộn theo cả trang), dùng đặc tính background-attachment.

body { background-image: url("img_tree.jpg"); 
background-repeat: no-repeat; 
background-position: right top; 
background-attachment: fixed;}

Ảnh nền, đặc tính rút gọn

Để rút ngắn code, bạn có thể chỉ định tất cả các đặc tính cho nền trong một đặc tính duy nhất, đây gọi là đặc tính rút gọn. Đặc tính rút gọn của nền là background.

body { background: #ffffff url("img_tree.jpg") no-repeat right top;}

Khi dùng đặc tính rút gọn, thứ tự các giá trị sẽ là

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

Lưu ý: Chỉ cần viết theo đúng thứ tự, không nhất thiết phải đầy đủ các đặc tính.

Đường viền


Thuộc tính border trong CSS giúp bạn xác định style, độ rộng và màu của đường viền bao quanh một phần tử. Tương ứng, bạn sẽ có ba thuộc tính sau giúp bạn thực hiện việc này:

  • Thuộc tính border-color xác định màu của đường viền.
  • Thuộc tính border-style giúp bạn xác định xem đường viền nên là solid, dashed, double, …
  • Thuộc tính border-width giúp bạn xác định độ rộng của đường viền.

Thuộc tính border-color trong CSS

Để xác định màu cho đường viền trong CSS, bạn sử dụng thuộc tính border-color. Ngoài việc xác định màu cho toàn bộ đường viền bao quanh phần tử, bạn cũng có thể xác định riêng rẽ màu cho từng phần của đường viền, chẳng hạn như phần trên, đáy, cạnh trái hoặc cạnh phải. Tùy theo mục đích, bạn sử dụng một trong các thuộc tính sau đây.

  • Thuộc tính border-bottom-color giúp bạn thay đổi màu của đáy đường viền.
  • Thuộc tính border-top-color giúp bạn thay đổi màu của phần trên đường viền.
  • Thuộc tính border-left-color giúp bạn thay đổi màu của cạnh trái đường viền.
  • Thuộc tính border-right-color giúp bạn thay đổi màu của cạnh phải đường viền.

Dưới đây là ví dụ minh họa cách sử dụng các thuộc tính trên để thay đổi màu của đường viền trong CSS:

        }
    </style>
</head>
    <body>
        <p class="example1">

        </p> 
        <p class="example2">
            Vi du hien thi Border voi mau duy nhat tren tat ca cac canh.
        </p>
    </body>
 </head>

Lề

Định nghĩa và cách sử dụng

Thuộc tính này margin thiết lập lề cho một phần tử và là thuộc tính viết tắt của các thuộc tính sau:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Ví dụ:

1. Nếu thuộc tính margin có bốn giá trị:

margin:10px 5px 15px 20px;
  • top margin: 10px
  • right margin: 5px
  • bottom margin: 15px
  • left margin: 20px

2. Nếu thuộc tính margin có ba giá trị:

margin:10px 5px 15px;
  • top margin: 10px
  • right and left margin: 5px
  • bottom margin: 15px

3. Nếu thuộc tính margin có hai giá trị:

margin:10px 5px;
  • top and bottom margin: 10px
  • right and left margin: 5px

4. Nếu thuộc tính margin có một giá trị:

margin:10px;
  • all four margins: 10px

Lưu ý: Giá trị âm được chấp nhận.


Cú pháp CSS

padding: margin: length|auto|initial|inherit;
Value Description
length Chỉ định khoảng đệm theo px, pt, cm, v.v. Giá trị mặc định là 0.
% Chỉ định khoảng đệm theo px, pt, cm, v.v. Giá trị mặc định là 0.
auto Trình duyệt tính toán biên độ
initial Đặt thuộc tính này về giá trị mặc định.
inherit Kế thừa thuộc tính này từ phần tử cha của nó.

Sụp đổ lề

- Lề trên và lề dưới của các phần tử đôi khi được thu gọn thành một lề duy nhất bằng lề lớn nhất trong hai lề.

- Điều này không xảy ra với lề ngang (trái và phải)! Chỉ xảy ra với lề dọc (trên và dưới)!

p.a {
  margin: 30px 0;
}

p.b {
  margin: 20px 0;
}

Ví dụ:

  • Trong ví dụ trên, phần tử <p class="a"> có lề trên và lề dưới là 30px. Phần tử <p class="b"> có lề trên và lề dưới là 20px.
  • Điều này có nghĩa là lề dọc giữa <p class="a"> và <p class="b"> phải là 50px (30px + 20px). Nhưng do việc thu gọn lề, lề thực tế lại là 30px!

Ví dụ

Đặt lề cho phần tử <p> thành 35 pixel cho phần trên và phần dưới, và thành 70 pixel cho phần bên phải và bên trái:

p {
  margin: 35px 70px;
}

Đặt lề cho phần tử <p> thành 35 pixel cho phần trên, 70 pixel cho phần bên phải và bên trái, và 50 pixel cho phần dưới:

p {
    marging: 35px 70px 50px;
}

Đặt khoảng đệm cho phần tử <p> thành 35 pixel cho phần trên, 70 pixel cho phần bên phải, 50 pixel cho phần dưới và 90 pixel cho phần bên trái:

p {
    margin: 35px 70px 50px 90px;
}

Định nghĩa và cách sử dụng

  • padding-bottom
  • padding-left

Lưu ý: Padding tạo thêm khoảng trống bên trong một phần tử, trong khi margin tạo thêm khoảng trống xung quanh một phần tử.

Thanh điều hướng


  • Có một hệ thống điều hướng dễ sử dụng là rất quan trọng đối với bất kỳ trang web nào!
  • Thanh điều hướng CSS là một thành phần quan trọng của thiết kế web. Thanh điều hướng giúp người dùng dễ dàng điều hướng giữa các phần khác nhau của trang web.
  • Thanh điều hướng thường được xây dựng bằng các phần tử danh sách HTML (<ul> và <li>), sau đó được định dạng bằng CSS để có giao diện đẹp mắt.
  • Thanh điều hướng thường nằm ở đầu hoặc bên cạnh trang web.

Thanh điều hướng = Danh sách liên kết

  • Thanh điều hướng cần có HTML chuẩn làm cơ sở.
  • Thanh điều hướng về cơ bản là một danh sách các liên kết, do đó việc sử dụng các phần tử <ul> và <li> là hoàn toàn hợp lý:

Ví dụ:

<ul>
  <li><a href="default.asp">Home</a></li>
  <li><a href="news.asp">News</a></li>
  <li><a href="contact.asp">Contact</a></li>
  <li><a href="about.asp">Abou</a></li>
</ul>

Bây giờ chúng ta hãy xóa các dấu đầu dòng, lề và phần đệm khỏi phần tử <ul>:

Ví dụ:

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

Ví dụ được giải thích:

  • Đặt list-style-type: none; - Xóa các dấu đầu dòng khỏi danh sách
  • Đặt margin: 0; - Đặt lại lề trình duyệt mặc định
  • Đặt padding: 0;- Đặt lại phần đệm mặc định của trình duyệt

Lưu ý: Mã HTML và CSS trong ví dụ trên là mã cơ sở được sử dụng cho cả thanh điều hướng dọc và ngang.


Thanh điều hướng dọc CSS

  • Trang chủ
  • Tin tức
  • Liên hệ
  • Về
  • Trong thanh điều chỉnh hướng dọc, các liên kết điều hướng được sắp xếp theo chiều dọc (đồng lên nhau) và thường được điều chỉnh theo chiều dọc theo bên trái hoặc bên phải của trang web.
  • Cơ sở của thanh điều hướng dọc là danh sách không có thứ tự (<ul>), với các danh sách mục (<li>), mỗi mục chứa một liên kết (<a>).

Ví dụ:

Tạo một cơ sở dọc điều hướng cho màu nền xám và chúng tôi cũng thay đổi màu nền và chữ màu của các liên kết khi người dùng di chuột qua chúng

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  width: 200px;
  background-color: #f1f1f1;
}

li a {
  display: block;
  color: black;
  padding: 8px 16px;
  text-decoration: none;
}

/* Change the link and background color on hover */
li a:hover {
  background-color: #555555;
  color: white;
}

Kết quả:

  • Trang chủ
  • Tin tức
  • Liên hệ
  • Về
  • Khối đầu tiên (ul) tương tự như trang trước. Ngoài ra, chúng tôi đã chỉ định cố định chiều rộng (200px) và màu nền xám nhạt.
  • Khối tiếp theo (li a) các phần tử <a> được định nghĩa bên trong các phần tử <li>.
  • Hiển thị các liên kết dưới dạng phần tử giúp toàn bộ liên kết vùng có thể nhấp (không chỉ văn bản) và cho phép chúng tôi xác định chiều rộng, khoảng đệm, cột, chiều cao, v.v. Chúng tôi thêm liên kết màu và một số khoảng đệm. Chúng tôi cũng xóa bỏ phần tử <a> này.
  • Khối cuối cùng (li a:hover) thay đổi màu liên kết và màu nền khi người dùng di chuột qua chúng.

Trạng thái hoạt động

Bây giờ chúng ta thêm một lớp " hoạt động " để làm liên kết nổi bật tương ứng với trang hiện tại để cho người dùng biết họ đang ở trang/phần nào.

.active {
  background-color: #04AA6D;
  color: white;
}

Liên kết thanh điều hướng trung tâm & Thêm đường viền

  • Để điều hướng giữa các liên kết, hãy thêm text-align: center; vào <li> hoặc <a>.
  • Nếu bạn muốn có đường viền xung quanh thanh điều hướng, hãy thêm thuộc tính border vào <ul>.
  • Nếu bạn cũng muốn có đường viền bên trong thanh điều hướng, hãy thêm thuộc tính border-bottom vào tất cả các phần tử <li>, ngoại trừ phần tử cuối cùng:

Ví dụ:

ul {
  border: 1px solid #555555;
}

Thanh điều hướng ngang CSS

  • Trang chủ
  • Tin tức
  • Liên hệ
  • Về
  • Trong điều chỉnh hướng ngang, các liên kết điều hướng được xếp chồng lên theo chiều ngang (cạnh nhau) và thường được điều chỉnh ở đầu trang web.
  • Cơ sở của thanh điều hướng ngang là danh sách không có thứ tự (<ul>), với các danh sách mục (<li>), mỗi mục chứa một liên kết (<a>).
  • Người ta cũng thường thêm phần tử <nav> vào phần tử <ul> để đóng vai trò trò chơi như một vùng chứa điều hướng của bạn.

Thanh điều hướng ngang sử dụng Float

Một cách để tạo thanh điều hướng ngang là thêm thuộc tính float vào phần tử <li>.

Ở đây chúng ta tạo một thanh điều hướng ngang cơ bản với màu nền tối và thay đổi màu nền của các liên kết khi người dùng di chuột qua chúng:


Ví dụ:

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333333;
}

ul li {
  float: left;
}

ul li a {
  display: block;
  color: white;
  text-align: center;
  padding: 14px 16px;
  text-decoration: none;
}

ul li a:hover {
  background-color: #111111;
}

  • Tin tức
  • Liên hệ
  • Về

Giải thích:

  • overflow: hidden; - Ngăn các mục danh sách ra khỏi danh sách
  • background-color: #333333; - Thêm màu nền đen vào phần tử <ul>
  • float: left; - Làm cho các phần tử <li> nổi cạnh nhau
  • display: block; - Cho phép chúng ta chỉ định phần đệm, chiều cao, chiều rộng và lề cho <a>
  • padding: 14px 16px; - Thêm một số khoảng đệm giữa mỗi phần tử <a>.
  • text-decoration: none; - Xóa gạch chân khỏi các liên kết

Thanh điều hướng ngang sử dụng Flex

Một cách hiện đại hơn để tạo thanh điều hướng ngang là sử dụng CSS flexbox.

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  background-color: #333333;
  display: flex;
}

ul li a {
  display: block;
  color: white;
  padding: 14px 16px;
  text-decoration: none;
}

ul li a:hover {
  background-color: #111111;
}
  • Giải pháp ở đây là display: flex;. Lệnh này tạo ra một ngữ cảnh flex và theo mặc định, các mục được hiển thị từ trái sang phải.
  • Và chỉ cần thêm một dòng vào khối ul: justify-content: center; , bạn sẽ có thanh điều hướng được căn giữa theo chiều ngang:

Ví dụ: Thanh điều hướng nằm ngang ở giữa với flex

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  background-color: #333333;
  display: flex;
  justify-content: center;
}
