HTML & CSS Tutorial

1) HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu để xây dựng cấu trúc trang web.

<!DOCTYPE html>
<html lang="vi">
  <head>
    <meta charset="UTF-8" />
    <title>Ví dụ HTML</title>
  </head>
  <body>
    <h1>Tiêu đề</h1>
    <p>Đoạn văn bản.</p>
  </body>
</html>

2) CSS là gì?

CSS (Cascading Style Sheets) dùng để trình bày: màu sắc, font, bố cục.

h1 {
  color: #1f6feb;
}
p {
  line-height: 1.6;
}

3) Flexbox nhanh

.row {
  display: flex;
  gap: 16px;
}
.col {
  flex: 1;
}

4) Grid nhanh

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
